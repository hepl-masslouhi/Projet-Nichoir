Projet de Nichoir Connecté

SmartCities 2025-2026

Suivi intelligent de la vie des oiseaux avec un coût accessible et une autonomie énergétique prolongée.

Proposition de Valeur

  • Prix : moins de 50 €
  • Autonomie énergétique : 6 mois à 1 an sur batterie
  • Évolutif : possibilité d’ajouter un panneau solaire
  • Suivi en temps réel via interface Web et MQTT

Caractéristiques Techniques

Capteur de Présence

PIR pour détecter les oiseaux.

Caméra TimerCam

OV3660, 3MP, DFOV 66.5°, résolution jusqu’à 2048x1536.

Raspberry Pi

Serveur Web Flask, stockage et traitement des données via MQTT.

Gestion d'Énergie

Ultra-basse consommation, RTC intégré, batterie externe possible.

Fonctionnement du Système

  1. Capteur PIR détecte la présence d’oiseaux.
  2. TimerCAM capture une photo avec éclairage IR.
  3. Photo envoyée au Raspberry Pi via Wi-Fi.
  4. Données reçues par script Python et stockées dans MariaDB.
  5. Interface Web Flask pour visualiser galerie et niveau de batterie.
